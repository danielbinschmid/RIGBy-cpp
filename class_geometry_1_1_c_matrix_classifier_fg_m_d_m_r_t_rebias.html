Classify the matrix and return the class id, the distance and the probability of each class.

  1. Transform the sample to the Tangent Space.
  2. Apply the FgDA weight.
  3. Return to the original Manifold.
  4. Apply the classify function of MDM Classifier (see CMatrixClassifierMDM::classify)

Remark : We use the MDM classification whatever the adaptation method chosen. Thus the MDM part evolves but the geodesic filtering does not evolve to keep an execution online. A version allowing the adaptation of the Filter will be implemented for offline execution.

Parameters
sampleThe sample to classify.
classIdThe predicted class.
distanceThe distance of the sample with each class.
probabilityThe probability of the sample with each class.
adaptationAdaptation method for the classfier EAdaptations.
realClassIdThe expected class id if supervised adaptation.
Returns
True if it succeeds, False otherwise.
